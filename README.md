# AWS Docker Deployment

A DevOps project focused on deploying a Dockerized web application on an AWS EC2 Linux instance. The project demonstrates containerization, cloud deployment, Linux server management, and basic DevOps practices.

---

## Features

- Dockerized web application deployment
- AWS EC2 instance setup and configuration
- Linux server management
- Port mapping and container execution
- Secure remote access using SSH
- Basic deployment workflow

---

## Technologies Used

- Docker
- AWS EC2
- Linux
- Git & GitHub

---

## How to Run

1. Launch an AWS EC2 Linux instance
2. Install Docker on the server
3. Clone the repository
4. Build the Docker image
5. Run the Docker container
6. Access the application using the EC2 public IP

---

- AWS cloud deployment basics

- Remote server management
